Huginn Net is a Rust library for passive fingerprinting and analysis of multiple network protocols, including TCP, HTTP, and TLS.
Inspired by p0f, Huginn Net extends passive OS fingerprinting to support modern detection standards and multi-protocol analysis.
It leverages pnet, httparse, and pcap-file to analyze raw packets and extract protocol-specific features for robust, non-intrusive network monitoring.
Project is under active development. Feedback and contributions welcome.

Network Stack (OSI Model)

Layer Name Protocol / Feature Example in Huginn Net

7 Application Layer HTTP Headers, User-Agent, Lang
4 Transport Layer TCP OS Fingerprinting (p0f)

Getting Started

To get started with Cargo, simply add the following line to your Cargo.toml file.

[dependencies]
huginn-net = "1.4.0"

For changes, refer to the release notes.
